# 🎵 Spotify Pulse Dashboard

## 📌 Project Overview

The Spotify Pulse Dashboard is an interactive data analytics project developed using Power BI to analyze Spotify streaming data and listener behavior. The dashboard provides insights into top artists, trending songs, genre popularity, streaming patterns, and audience engagement through dynamic visualizations and KPI tracking.

This project focuses on converting raw music streaming data into meaningful insights using data visualization and business intelligence techniques.

---

# 🎯 Objectives

* Analyze music streaming trends and listener behavior
* Identify top-performing artists and songs
* Track genre popularity and audience engagement
* Create interactive dashboards for better data exploration
* Generate insights using data-driven visualizations

---

# 🛠️ Tools & Technologies Used

* Microsoft Power BI
* Microsoft Excel
* DAX (Data Analysis Expressions)
* Data Cleaning & Transformation
* Data Visualization

# 📊 Dashboard Features

* Top Artists Analysis
* Most Streamed Songs
* Genre Popularity Insights
* Listening Trend Analysis
* Interactive Filters & Slicers
* KPI Cards & Visual Reports

# 📈 Key Insights

* Analyzed listener engagement and music consumption trends

* Observed peak streaming behavior and audience preferences
